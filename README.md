# 📖 日记助手 Diary Assistant

通过微信发送碎片化生活动态，AI 自动附带时间戳收录，每日清晨汇总为第一人称日记，按年/月分目录存档。

## 快速开始

初次启动时 AI 会通过对话问你几个问题，自动完成配置：

```
> 怎么称呼你？     → 老头
> 日记存哪个文件夹？→ D:\03\日记\  (回车默认)
> 用微信还是 Telegram？ → 微信
```

配好后直接发消息：

```
你：起床啦       → AI：已记下 ✅ 07:30
你：午饭         → AI：已记下 ✅ 12:15
你：下午去办了事  → AI：已记下 ✅ 15:00
```

次日清晨自动生成日记。

| 平台 | 方式 |
|------|------|

| **Claude Code** | 参考 `core/` 手动配置 |
| **Cursor** | 参考 `core/` 手动配置 |

## 核心功能

- 📩 消息自动收录 + 时间戳
- 🧠 内容含时间则优先使用（事后补记）
- 🌙 每天 06:00 自动汇总
- 🗣️ 说「收工」「睡了」即刻汇总
- 📂 按年月归档 `YYYY/MM/YYYY-MM-DD.md`
- ✍️ 原汁原味，不净化不删减
