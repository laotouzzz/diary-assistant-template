# 📖 日记助手 Diary Assistant

通过微信（或其他聊天软件）发送碎片化生活动态，AI 自动附带时间戳收录，每日清晨汇总为第一人称日记，按年/月分目录存档。

## 快速体验

初次启动时，AI 会通过对话问你几个问题：

```
> 怎么称呼你？
> 老头

> 日记存哪个文件夹？（默认 D:\03\日记\）
> 

> 用微信还是 Telegram？
> 微信
```

回答完自动完成全部配置，然后直接发消息就能用了：

```
你：起床啦
AI：已记下 ✅ 07:30

你：午饭
AI：已记下 ✅ 12:15

你：下午三点去办了件事
AI：已记下 ✅ 15:00
```

次日清晨自动生成日记：

```markdown
# 2026-05-25 日记

07:30 今天起得不算晚，醒了就爬起来了。

12:15 中午随便吃了点午饭，填饱肚子就行。

15:00 下午去办了件事。
```

## 核心功能

- 📩 **消息收录** — 每条消息附带发送时间戳
- 🧠 **时间优先级** — 消息内容含时间则优先使用（支持事后补记）
- 🌙 **自动汇总** — 每天清晨 06:00 整理成第一人称日记
- 🗣️ **手动触发** — 说「收工」「睡了」即刻汇总
- 📂 **按年月归档** — `YYYY/MM/YYYY-MM-DD.md`
- ✍️ **原汁原味** — 脏话粗口口语照实记录，不净化

## 文件结构

```
日记目录/
├── SOUL.md              # 助手身份设定
├── USER.md              # 用户偏好信息
├── memory/
│   └── FACT.md          # 工作流持久知识
├── raw/
│   └── YYYY-MM-DD.md    # 白天暂存记录
├── YYYY/
│   └── MM/
│       └── YYYY-MM-DD.md # 成品日记
```
