# Changelog

## v1.0.0 (2026-05-25)

### 🎉 初始版本

- **交互式初始化**：首次启动对话式问答，自动配置身份、路径、通道
- **消息收录**：微信消息自动附带时间戳，支持事后补记（内容时间优先）
- **时间优先级**：消息内容含时间则优先使用，而非发送时刻
- **自动汇总**：每天清晨 06:00 强制汇总前一日日记
- **手动触发**：说「收工」「睡了」即刻汇总
- **按年月归档**：`YYYY/MM/YYYY-MM-DD.md`
- **原汁原味**：脏话粗口口语照实记录，不净化不删减
- **口吻学习**：根据历史原文校准风格，越来越像用户本人说话
- **后续修改命令**：`改设置` `改路径` `改称呼` `改时区` `改规则` `查看规则`
- **跨平台通用文档**：`core/` 目录适配 Claude Code、Cursor 等平台
- **安装脚本**：`scripts/setup.ps1` 一键初始化 (PowerShell)
